# NEON DISTRICT — Art Direction

## Direction

Experimental digital-culture festival built as a living interface rather than a conventional event landing page.

## Visual system

- Deep black base with neon magenta, cyan, acid lime and electric violet
- Compressed display typography with outlined and displaced layers
- Canvas-based spatial tunnel and signal rings
- Festival posters designed entirely in CSS
- Hard grid, noisy texture and sharp transitions
- Responsive transformation from cinematic desktop scenes to touch-first vertical storytelling

## Core interaction

The site behaves like a signal network. Visitors switch zones, mutate posters, change the lineup day, move through a time tunnel and select an access depth.
